Flutter.io

  • Mobile app SDK developed by Google
  • Cross platform compatible code for Android and iOS
  • Provides a rich set of Material Design and Cupertino style widgets
  • Everything is a Widget
  • Example: Structural elements(i.e. buttons or menus) Stylistic elements(i.e. fonts or colors)
  • Hierarchy: Each widget nests inside
  • "Stateless" and "Stateful" widgets
