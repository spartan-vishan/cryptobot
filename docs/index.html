Flutter.io

  • Mobile app SDK developed by Google
  • Cross platform compatible code for Android and iOS
  • Provides a rich set of Material Design and Cupertino style widgets
  • Everything is a Widget
  • Example: Structural elements(i.e. buttons or menus) Stylistic elements(i.e. fonts or colors)
  • Hierarchy: Each widget nests inside
  • "Stateless" and "Stateful" widgets

Dart

  • General-purpose language developed by Google
  • While it is technically an object-oriented language, it supports functional programming constructs.
  • Suitable for both mobile and web apps

Benefits of Dart

  • Can be both object oriented and functional
  • Dart is strongly typed. Uses abstract classes and static typing
  • Familiar C-style syntax
  • In web browser, Dart code can be compiled as JavaScript. As such it is compatible with all major browsers wth no need for browsers to adopt Dart
  • Single codebase for both iOS and Android
  • Dart VM Hot Reload allows for fast development cycles
  • Rich, well documented library

Firebase Backend

  • Mobile app SDK developed by Google as part of the Google cloud
  • Easy to implement database platform for mobile dev and web programming
  • Allowed us to implement user authentication via a users Google profile
